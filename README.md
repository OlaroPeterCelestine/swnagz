# Swangz Avenue Website

A modern music production and events website featuring talent showcases, event calendars, and news updates.

## Features

- **Hero Section** with background video
- **Our Talent** showcase grid
- **Event Calendar** with upcoming events
- **News Section** with featured articles
- **Responsive navigation** with dropdown menus
- **Social media integration** with Font Awesome icons

## Tech Stack

- HTML5
- Tailwind CSS
- Font Awesome Icons
- Google Fonts (Montserrat)

## Deployment Instructions

### 1. Push to GitHub

First, create a new repository on GitHub, then run:

```bash
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

### 2. Deploy to Vercel

1. Go to [vercel.com](https://vercel.com)
2. Sign in with your GitHub account
3. Click "Add New Project"
4. Import your Swangz Avenue repository
5. Vercel will automatically detect the settings and deploy
6. Your site will be live at `your-project.vercel.app`

### What Vercel Does

- Automatically builds and deploys your site
- Provides a custom domain
- SSL certificate included
- Automatic deployments on every push to main branch
